# Landing Page Template (HTML/CSS + minimal JS)

A small, review-friendly landing page built with **plain HTML + CSS** and a little **JavaScript**.

## What this demonstrates
- **Semantic HTML** structure and clean sectioning
- **Responsive layout** (grid/flex, mobile-first breakpoints)
- **Reusable UI components** (buttons, cards, simple spacing system)
- **JavaScript affinity**: mobile nav toggle + FAQ accordion
- **Accessibility basics**: focus-visible styling + ARIA state updates

## How to run
Open `index.html` in a browser.
